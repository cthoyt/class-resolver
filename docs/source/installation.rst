Installation
============

The most recent release can be installed from `PyPI
<https://pypi.org/project/class_resolver>`_ with:

.. code-block:: shell

    $ pip install class_resolver

The most recent code and data can be installed directly from GitHub with:

.. code-block:: shell

    $ pip install git+https://github.com/cthoyt/class-resolver.git

To install in development mode, use the following:

.. code-block:: shell

    $ git clone git+https://github.com/cthoyt/class-resolver.git
    $ cd class-resolver
    $ pip install -e .
